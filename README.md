# Multi-Agent AI Productivity Assistant

A Python-based multi-agent AI assistant to help manage **tasks, notes, and schedules**. Built with **Streamlit** for an interactive web interface.

---

## 🚀 Features

- **Task Management** – Add, view, and track tasks.  
- **Notes Management** – Take notes and view them instantly.  
- **Schedule Management** – Add and view schedule items.  
- **Multi-Agent Architecture** – Separate agents handle tasks, notes, and schedules for modularity.

---

## 🛠️ Technologies Used

- Python 3.11  
- Streamlit (Web Interface)  
- Custom Agents: `ScheduleAgent`, `NotesAgent`, `TaskAgent`
